# Teslo API

1. Clone the repository
2. Run `yarn install`
3. Clone the `.env.example` file and rename it to `.env`
4. Change the values in the `.env` file to your own
5. Run the database:
```
docker-compose up -d
```
6. Run SEED:
```
https://localhost:3005/api/seed
```
7. Build the project:
```
yarn start:dev
```
